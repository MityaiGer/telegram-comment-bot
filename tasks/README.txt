-id: Уникальный идентификатор задачи.
-link: Ссылка на пост в Телеграм, к которому будут добавляться комментарии.
-num_comments: Общее количество комментариев, которые нужно отправить.
-comments: Массив с текстами комментариев.
-status: Статус задачи. Может быть "pending" (в ожидании), "in -progress" (в процессе) или "completed" (завершено).
-comments_sent: Количество комментариев, которые уже были отправлены. Это поле можно использовать для отслеживания прогресса выполнения задачи.
